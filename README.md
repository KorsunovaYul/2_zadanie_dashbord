# Умный Дашборд — персональная панель инструментов

## Описание проекта

Это интерактивный веб-дашборд с динамическими виджетами, позволяющий пользователям создать персонализированную рабочую панель с различными инструментами и мини-играми. Проект демонстрирует работу с API, динамическое создание элементов, управление состоянием и использование localStorage для сохранения настроек.

## Функциональность

- **Динамические виджеты** - добавление, удаление и настройка виджетов в реальном времени
- **Drag & Drop** - перетаскивание виджетов для изменения их расположения
- **Автосохранение** - конфигурация дашборда автоматически сохраняется в localStorage
- **Импорт/Экспорт** - возможность экспортировать и импортировать конфигурацию в формате JSON
- **API интеграция** - подключение к внешним API для получения актуальных данных
- **Адаптивный дизайн** - полная адаптация под все размеры экранов
- **Интерактивные игры** - встроенные мини-игры для развлечения

## Доступные виджеты

### Информационные виджеты

1. **Погода** - актуальная информация о погоде в выбранном городе
   - Температура и ощущаемая температура
   - Влажность, давление, скорость ветра
   - Визуальное отображение погодных условий
   - Данные из API OpenWeatherMap

2. **Курсы валют** - текущие курсы валют к рублю
   - Данные от ЦБ РФ (CBR API)
   - Отображение динамики изменения курса
   - Поддержка USD, EUR и других валют
   - Цветовая индикация роста/падения

3. **Случайная цитата** - вдохновляющие цитаты
   - API Forismatic
   - Кнопка для получения новой цитаты
   - Отображение автора цитаты

### Продуктивные виджеты

4. **Таймер Pomodoro** - техника управления временем
   - Режимы работы (25 минут) и перерыва (5 минут)
   - Управление: старт, пауза, сброс
   - Переключение между режимами
   - Браузерные уведомления при завершении таймера

5. **Заметки** - быстрые записи
   - Добавление и удаление заметок
   - Хранение в localStorage
   - Быстрый ввод с Enter

### Игровые виджеты

6. **Ударь рыбку** (Whack-a-Fish) - игра в стиле Whack-a-Mole
   - 9 лунок с появляющимися рыбками
   - Ограничение по времени (30 секунд)
   - Подсчет очков
   - Визуальные эффекты и анимации

7. **Морская память** (Memory Match) - игра на память
   - Поиск парных морских существ
   - 12 карточек (6 пар)
   - Подсчет ходов
   - Плавные анимации переворота карт

8. **Охотник за сокровищами** (Treasure Hunt) - игра-собиратель
   - Управление дайвером (стрелки клавиатуры)
   - Сбор монет, жемчуга и драгоценностей
   - Система кислорода (таймер)
   - Динамическое появление сокровищ

## Технологии

## Реализованный JavaScript функционал

### Управление виджетами
```javascript
// Добавление виджета с автосохранением
addWidget(type, config = {}) {
    const widgetId = `widget-${this.nextWidgetId++}`;
    const widget = { id: widgetId, type: type, config: config, data: null };
    this.widgets.set(widgetId, widget);
    this.render();
    this.saveToStorage();
    this.loadWidgetData(widgetId);
}
```

### Работа с внешними API

## API интеграции

### OpenWeatherMap API
- **Назначение:** Получение данных о погоде
- **Endpoint:** `https://api.openweathermap.org/data/2.5/weather`
- **Параметры:** city (город), units=metric (метрическая система), lang=ru (русский язык)

### ЦБ РФ API
- **Назначение:** Курсы валют
- **Endpoint:** `https://www.cbr-xml-daily.ru/daily_json.js`
- **Формат:** JSON с актуальными курсами всех валют

### Forismatic API
- **Назначение:** Случайные цитаты
- **Endpoint:** `https://api.forismatic.com/api/1.0/`
- **Метод:** JSONP (для обхода CORS)

## Использование

1. Откройте файл `index.html` в современном браузере
2. Нажмите "Добавить виджет" для выбора нужных инструментов
3. Настраивайте виджеты через кнопку "Настройки" (⚙️)
4. Перетаскивайте виджеты для изменения их расположения
5. Используйте "Экспорт" для сохранения конфигурации
6. Импортируйте сохраненную конфигурацию через "Импорт"

### Горячие клавиши (игры)

- **Охотник за сокровищами:** ← → ↑ ↓ для управления дайвером
- **Заметки:** Enter для добавления новой заметки

## Требования браузера

- Современный браузер с поддержкой ES6+ (Chrome 60+, Firefox 60+, Safari 12+, Edge 79+)
- Включенный JavaScript
- localStorage (для сохранения конфигурации)
- Notifications API (опционально, для уведомлений таймера)

## Возможности расширения

Проект легко расширяется новыми виджетами:

1. Добавьте новый тип в метод `getWidgetContent()`
2. Создайте метод рендеринга `render[ТипВиджета]Widget()`
3. Добавьте иконку и название в HTML (модальное окно)
4. Реализуйте загрузку данных в `loadWidgetData()`
